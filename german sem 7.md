|           | einem | einr | einem | -      |

## Rules
- **Nominativ** for subject
- **Akkusativ** for direct object
- **Dativ** for indirect object

## examples
- Das ist ein Kuli
	- here *Kuli* is subject
- Der Lehrer hat einen Kuli
	- here *Lehrer* is subject, *Kuli* is object
